# 好用工具

## 文件处理网站

- [ILOVEPDF](https://www.ilovepdf.com/zh-cn)：包含对`pdf`合并、拆分、压缩、转换格式。编辑等多个功能。
- [ALL TO ALL](https://www.alltoall.net/)： 不同文件在线格式转换，支持所有的视频、音频、图片文件等等。
- [cloudconvert](https://cloudconvert.com/) ：文件格式转换
- [Convertio](https://convertio.co/zh/)：在线文件转换器（就是有时识别精度不高）

## 图片处理网站

- [IMAGE CYBORG](https://imagecyborg.com/) ：可以一键获取某个网站内所有图片（输入网站地址就可以一键下载所有图片）

- [bigjpg](https://bigjpg.com/)：图片放大自适应

- [tiny png](https://tinypng.com/)：图片压缩

- [Squoosh](https://squoosh.app/) ：缩小限制，可以自由自定义压缩

- [Recompressor](https://recompressor.com/)：更加可视化查看图片压缩过程

- [Yandex](https://yandex.com/images/)：搜图网站，可以根据图片，搜索出相关大小，相似图片

- [ILOVEIMG](https://www.iloveimg.com/)：图片编辑压缩等等功能

- 抠图：

  - [bgeraser](https://bgeraser.com/)：每月**五张**免费，抠图质量好

  - [removebg](https://www.remove.bg/)：抠图质量最高，但**一张**免费
  - [稿定抠图](https://koutu.gaoding.com/)：三张免费。。。
  - [顽兔抠图](https://alidesign.taobao.com/work.htm#)：抠图质量差，但是无限制

## 文字处理类网站

- [OCRSpace](https://ocr.space/) ：可以根据图片识别图片内所有文字（识别精度高，支持多语言）
- [白描](https://web.baimiaoapp.com/)：功能窗口简洁，还可以识别数字公式和电子表格
- [全能扫描王](https://www.camscanner.com/)：识别精度高，但需要登录才能使用
- [OCRMaker](http://www.ocrmaker.com/)：识别精度高，支持多语言
- [网易见外工作台](https://jianwai.youdao.com/)：可以将语音视频转换成文字

## 综合工具类网站

- [在线工具箱](https://www.zxgj.cn/)：多功能小工具
- [一个工具箱](http://www.atoolbox.net/)：三百多个功能，集成大部分工具

## 其他工具

- [GifCam](http://blog.bahraniapps.com/gifcam/)：gif 制作工具，小巧免费
- [avataaars](https://getavataaars.com/)：可以生成有趣的拟人化头像
- [CSS Peeper](https://csspeeper.com/)：CSS Peeper 是一款 Chrome 拓展程序，能帮助检查不同网页的样式信息，展示网站所使用的颜色和字体，比如我们在某个网站看到某一个元素，可以使用它快速的帮助我们获取它的 CSS 样式。
- [Happy Hues](https://www.happyhues.co/)：UI 配色网站设计

## 开发工具

- [brumm.af](https://shadows.brumm.af/)：通过使用这款工具，你可以创造出惊人的 `box-shadow` 阴影效果。你可以添加多个层级的阴影，并且对每个图层进行调整，从而创造出最自然、最拟真的阴影效果。
